Guia Rápido: Configurando o Firebase para seu Aplicativo
Siga estes passos para criar seu projeto na nuvem e conectar ao aplicativo.

Passo 1: Criar um Projeto no Firebase
Acesse o Console do Firebase.

Clique em "Adicionar projeto" (ou "Create a project").

Dê um nome ao seu projeto (ex: app-suporte-n2) e aceite os termos.

Desative a opção do Google Analytics para este projeto, se preferir (não é necessário para esta aplicação).

Clique em "Criar projeto" e aguarde a conclusão.

Passo 2: Adicionar o Aplicativo Web ao Projeto
No painel do seu novo projeto, clique no ícone da Web, que se parece com </>.

Dê um "Apelido" ao seu aplicativo (ex: App Suporte Web).

NÃO marque a opção "Configurar também o Firebase Hosting".

Clique em "Registrar aplicativo".

Passo 3: Copiar as Credenciais (O Passo Mais Importante!)
Após registrar, o Firebase mostrará um código JavaScript dentro de uma caixa com o título "Adicionar o SDK do Firebase".

Dentro desse código, você encontrará um objeto chamado firebaseConfig.

Copie todo o conteúdo desse objeto, desde { até }.

Abra o seu arquivo script.js, encontre a seção de configuração do Firebase no topo e cole o objeto que você copiou, substituindo o texto de exemplo (SUA_API_KEY, etc.).

// O seu código deve ficar parecido com isto:
const firebaseConfig = {
  apiKey: "AIzaSy...xxxxxxxxxxxx",
  authDomain: "app-suporte-n2.firebaseapp.com",
  projectId: "app-suporte-n2",
  storageBucket: "app-suporte-n2.appspot.com",
  messagingSenderId: "1234567890",
  appId: "1:1234567890:web:xxxxxxxxxxxx"
};

Após copiar, clique em "Continuar no console".

Passo 4: Habilitar a Autenticação Anônima
No menu esquerdo do Firebase, vá para Authentication.

Clique no botão "Primeiros passos".

Na nova tela, vá para a aba "Sign-in method".

Na lista de provedores, clique em "Anônimo" e ative a opção no botão toggle. Salve.

Passo 5: Criar e Configurar o Banco de Dados (Firestore)
No menu esquerdo, vá para Firestore Database.

Clique em "Criar banco de dados".

Selecione "Iniciar em modo de produção" e clique em "Avançar".

Escolha uma localização para os seus dados (recomendo southamerica-east1 (São Paulo) por estar no Brasil) e clique em "Ativar".

Passo 6: Definir as Regras de Segurança
Ainda no Firestore, vá para a aba "Regras".

Substitua o conteúdo existente pelas seguintes regras:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Permite que um usuário leia e escreva apenas no seu próprio documento
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Clique em "Publicar". Essas regras garantem que os dados de um usuário só possam ser acessados por ele mesmo.

Pronto! Seu projeto Firebase está configurado. Ao recarregar seu aplicativo, ele irá se conectar automaticamente à nuvem, salvar e carregar seus dados de forma segura.
